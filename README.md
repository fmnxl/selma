Selma
=====

A terminal Snake implementation written in Haskell.

![alt text](https://github.com/freemanon/selma/raw/master/preview.gif "Selma")


# Requirements

## Build & Packaging

* GHC
* Cabal

## Haskell packages

* Brick

# Build

```sh
ghc -threaded -outputdir ./dist -o ./dist/Main ./Main.hs && ./dist/Main
```
